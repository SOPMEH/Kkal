# ккал — установка на телефон (себе и маме)

Это PWA-приложение: один раз выкладываешь его в интернет, потом на каждом телефоне
добавляешь на главный экран — и оно работает как обычное приложение со своей иконкой,
на весь экран и с сохранением всех данных прямо на телефоне.

## Шаг 1. Выложить на GitHub Pages (бесплатно, 5 минут)

1. Зайди на github.com → New repository → имя например `kkal` → Public → Create.
2. Add file → Upload files → перетащи ВСЕ файлы из этой папки:
   `index.html`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`, `apple-touch-icon.png`
   → Commit changes.
3. Settings → Pages → в разделе "Branch" выбери `main` и папку `/ (root)` → Save.
4. Через 1–2 минуты приложение будет жить по адресу:
   `https://ТВОЙ_НИК.github.io/kkal/`

## Шаг 2. Получить бесплатный AI-ключ (один раз)

Вариант А — Groq (у тебя уже есть ключ от Мили Советчика, он подойдёт):
- console.groq.com → API Keys → Create API Key → скопировать (начинается с `gsk_`).

Вариант Б — Google Gemini:
- aistudio.google.com → Get API key → Create API key → скопировать (начинается с `AIza`).

Оба бесплатны, лимитов хватает на десятки фото в день. В приложении: ⚙ → выбрать
сервис → вставить ключ → Сохранить. Ключ хранится только на этом телефоне.
На телефоне мамы нужно вставить ключ один раз так же (можно тот же самый).

## Шаг 3. Установить на телефоны

Android (Chrome):
1. Открыть ссылку из шага 1.
2. Меню ⋮ → «Добавить на главный экран» / «Установить приложение».

iPhone (Safari):
1. Открыть ссылку из шага 1 именно в Safari.
2. Кнопка «Поделиться» (квадрат со стрелкой) → «На экран "Домой"».

Готово: иконка «ккал» появится среди приложений, открывается на весь экран.

## Важно знать

- Данные дневника хранятся локально на каждом телефоне (localStorage):
  у мамы — её дневник, у тебя — твой. Никуда не отправляются.
- Если очистить данные браузера/Safari — дневник сотрётся.
- Фото еды отправляется только в Groq/Gemini для распознавания и нигде не сохраняется.
- Чтобы обновить приложение — просто загрузи новый index.html в тот же репозиторий.

## Если что-то не работает

- Фото-анализ выдаёт ошибку ключа → проверь, что ключ вставлен без пробелов
  и выбран правильный сервис (Groq-ключ начинается с gsk_, Gemini — с AIza).
- «Лимит запросов» → бесплатный лимит на минуту исчерпан, подожди и повтори.
- На iPhone приложение не ставится → ссылку нужно открывать в Safari, не в Chrome.
